  imem siblings <result-id>
  # = search(filters={'file_path': result.file_path})

  imem temporal <result-id>
  # = search(filters={'timestamp': f'>{result.timestamp}'}) + semantic threshold

  imem session <result-id>
  # = search(filters={'session_id': result.session_id})

  Question: Do we need special commands? Or expose metadata filtering directly?

  Tension:
  - Special commands = discoverable, loggable
  - Direct filtering = maximally flexible

  Minimal option:
  # Just expose powerful filter syntax
  imem filter --file-path <path>
  imem filter --session <id>
  imem filter --timestamp-after <ts> --semantic-similar-to <result-id>

  Claude Code composes. We log the filter compositions.
