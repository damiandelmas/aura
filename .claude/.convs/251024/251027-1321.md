  Our Conversation Today: The Primitive Model

  No object model. Instead: Primitives
  # Core operations
  imem search <query> --filters {...}     → result_set (JSON)
  imem filter <conditions>                → result_set (JSON)
  imem graph build <results>              → graph_id
  imem graph apply <id> <algorithm>       → ranked results
  imem inspect <result-id>                → metadata
  imem combine <result-sets>              → merged result_set

  No hardcoded modes. Instead: Compositions
  # EXPLAIN emerges from composition
  imem search "JWT" --decisions --limit 1
  imem filter --file-path <result.path>      # siblings
  imem filter --session <result.session>     # conversation

  # Captured as slash command after proven useful
  /.claude/commands/explain.md

  # I send JSON, IMEM executes internally
