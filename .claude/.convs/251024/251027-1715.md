  The Flippable Chunk Architecture

  Storage (both indexed):
  Qdrant vectors:
  ├─ chunk_abc: 251011-1200_auth.md (implementation)
  │   metadata: {superseded_by: chunk_xyz, has_pattern: True}
  │
  └─ chunk_abc_pattern: 251011-1200_auth.pattern.md (abstraction)
      metadata: {source_impl: chunk_abc, layer: pattern}

  Serving decision (runtime):
  def serve_chunk(chunk_id):
      chunk = qdrant.retrieve(chunk_id)

      if chunk.superseded and chunk.has_pattern:
          # Flip the board
          pattern_id = chunk.metadata['pattern_chunk_id']
          return qdrant.retrieve(pattern_id)

      return chunk  # Current or no pattern available

  def serve_full_resolution(chunk_id):
      # Force implementation view
      return qdrant.retrieve(chunk_id)

  No re-indexing needed. Just serve different chunk.
