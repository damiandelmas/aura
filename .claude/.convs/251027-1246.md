  The Answer

  I construct JSON. IMEM accepts JSON. Done.

  # From my perspective (Claude Code):
  imem_search({
    "queries": [
      {"text": "auth security", "filters": {"section_type": "Decisions"}},
      {"text": "auth impl", "filters": {"layer": "implementation"}},
      {"text": "auth patterns", "filters": {"layer": "pattern"}}
    ],
    "combine": true,
    "graph": {
      "algorithm": "centrality",
      "top": 10
    }
  })

  # Returns: Single JSON response with ranked results

  IMEM internally:
  - Receives JSON
  - Spawns 3 queries in parallel (async/threading)
  - Merges results
  - Builds graph
  - Applies algorithm
  - Returns single response

  One call. One response. All parallelism internal.
