● Available Filter Fields

  Core Filters:

  | Field      | Type   | Values              | Use Case                              |
  |------------|--------|---------------------|---------------------------------------|
  | chunk_type | string | 'message', 'patch'  | Filter messages vs code changes       |
  | role       | string | 'user', 'assistant' | Filter by who said it (messages only) |
  | session_id | string | UUID                | Filter to specific conversation       |
  | file_path  | string | /path/to/file.py    | Filter patches by file (patches only) |

  Metadata Filters:

  | Field            | Type      | Example              | Use Case                      |
  |------------------|-----------|----------------------|-------------------------------|
  | start_time       | timestamp | 2025-11-02T00:46:33Z | Filter by date/time           |
  | duration_minutes | float     | 223.9                | Long vs short sessions        |
  | message_count    | int       | 375                  | Session size                  |
  | source           | string    | 'conversation'       | Distinguish from context docs |
