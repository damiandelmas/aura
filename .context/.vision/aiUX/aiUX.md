# aiUX: Design for AI Agent Comprehension

The user is the AI agent. When developing this system, every decision—how sections are ordered, what metadata is exposed, which relationships are explicit—affects whether AI agents repeat failed approaches or make informed suggestions.

Structure is the interface. A template that returns "❌ Failed Branch (Don't Suggest)" with 36 continuation counts isn't rendering for humans to read—it's routing AI attention away from known failures toward working solutions. Test new features by querying: "Do I understand which approach failed, why it failed, and what worked instead?" If you can't tell, neither can future AI agents helping developers.
