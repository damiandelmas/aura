What IMEM Actually Is

Not: General document search for humans
Is: AI Coding Agent Long-Term Memory

Architecture:
Layer 0: Template-as-Schema (100% metadata compliance)
Layer 1: Qdrant (immutable storage)
Layer 2A: FlexGraph Primitives (siblings, genealogy, temporal)
Layer 2B: Graph Operations (PageRank, centrality)
Layer 3: Serve-Time Structure (relationship-labeled context for AI)

Key Properties:
- AI-to-AI communication (not human search)
- Template enforcement (Context, Solution, Rationale fields guaranteed)
- Compositional intelligence (FlexGraph does the heavy lifting)
- Immutable changelogs (archaeological precision)
- Pre-structured serving
