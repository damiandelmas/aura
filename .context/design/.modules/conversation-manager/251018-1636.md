Capability B: Conversation Querying (Use Cases 3 & 4)
What it does: Claude asks questions to past conversations mid-session
How it works:
python# MCP Tool
@tool
def query_past_conversation(session_id: str, question: str) -> str:
    """Ask a question to a past conversation"""
    
    # 1. Load conversation transcript
    transcript = load_jsonl(f"~/.claude/projects/*/{session_id}.jsonl")
    
    # 2. Format as context
    context_md = jsonl_to_markdown(transcript)
    
    # 3. Query via headless Claude
    response = subprocess.run([
        "claude", "-p", 
        f"Question: {question}\n\nContext from past conversation:\n{context_md}"
    ], capture_output=True)
    
    return response.stdout
```

The document doesn't address **context filtering**.
```
