## Feature Usefulness (Reality)

### ✅ WORKS & USEFUL NOW

#### 1. Siblings Discovery
**Status:** Fully functional, high value

**What it does:**
- Finds all sections from same changelog file
- Uses `file_path` metadata (already exists)

**Test results:**
- 17 siblings found per query
- Shows Decisions, Constraints, Patterns, Implementation from same doc
- **Solves your "missed related sections" feedback**

**Real value:** ⭐⭐⭐⭐⭐ (This alone justifies Phase 6-7)

**Example:**
```bash
imem compose '{"search": {"text": "JWT", "limit": 1}, "discovery": {"siblings": true}}'
```
Returns primary match + all 17 sections from same changelog.

**Status:** Works but limited scope

**Real value:** ⭐⭐ (Performance win, limited usefulness without multi-phase data)
