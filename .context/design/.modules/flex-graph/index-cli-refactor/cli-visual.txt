8. REFACTOR TARGET (PROPOSED SOLUTION)
================================================================================

NEW COMMAND TREE (Verb-Noun, Consistent):

imem
├─ index
│  ├─ develop [--force]           ← Index .context/develop/
│  ├─ design [--force]             ← Index .context/design/
│  ├─ document [--force]           ← Index .context/document/
│  ├─ conversations [--limit]      ← Index ~/.claude/
│  │  [--recent] [--min-size]
│  └─ context [--force]            ← Convenience: all phases
│     [--include-design]
│
├─ search <query> [<query>...]     ← Unified search
│  [--in develop|design|document|
│       conversations|context]
│  [--limit] [--sort-by] [--after]
│  [--split-terms] [--operator]
│
├─ service
│  ├─ start
│  ├─ stop
│  └─ status
│
├─ compose '<config>'              ← Keep as-is (complex)
├─ update [--vscode]               ← Convenience
├─ dedupe [--dry-run]              ← Maintenance
└─ status                          ← Project status

BENEFITS:
  ✅ Consistent verb-noun everywhere
  ✅ Discoverable: `imem index` lists phases
  ✅ Intuitive: English word order
  ✅ Extensible: Easy to add new phases
  ✅ Per-project: Separate collections
  ✅ Backwards compatible: old syntax still works
