# Work on v2

## What's Required

### 1. Environment Variables (Per Session)

**Add to shell rc file:**

```bash
# ~/.bashrc or ~/.zshrc

# IMEM v2 activation
imem-v2() {
    conda activate imem-v2
    export IMEM_CONTEXT_DIR=~/.context_v2
    export IMEM_QDRANT_PORT=6334
}

# IMEM v3 activation  
imem-v3() {
    conda activate imem-v3
    export IMEM_CONTEXT_DIR=~/.context_v3
    export IMEM_QDRANT_PORT=6335
}
```

**Usage:**
```bash
imem-v2  # Switches to v2 with isolated context
imem-v3  # Switches to v3 with isolated context
```

---

### 2. Initial Setup (One-Time)

```bash
# Create v2 environment
conda create -n imem-v2 python=3.12 -y
conda activate imem-v2
export IMEM_CONTEXT_DIR=~/.context_v2
export IMEM_QDRANT_PORT=6334
pip install -e ~/projects/fleet/hangar/code/aura/main/imem

# Create v3 environment
conda create -n imem-v3 python=3.12 -y
conda activate imem-v3
export IMEM_CONTEXT_DIR=~/.context_v3
export IMEM_QDRANT_PORT=6335
pip install -e ~/projects/fleet/hangar/code/aura/worktrees/sql-first/imem
```

**That's it.** No code changes needed.

## Does It Support Iterations/Versions/Subversions?

**✅ YES - Multiple Approaches:**

### A. Feature Branches (Git Worktrees)
```bash
# Create v3.1 experimental branch
git worktree add ../v3.1-hnsw sql-first
cd ../v3.1-hnsw
git checkout -b feature/hnsw-backend

# New environment for v3.1
conda create -n imem-v3.1 python=3.12 -y
conda activate imem-v3.1
export IMEM_CONTEXT_DIR=~/.context_v3.1
export IMEM_QDRANT_PORT=6336
pip install -e .
```

**Result:** v2, v3, v3.1 all isolated and runnable.

---

### B. Same Environment, Different Data
```bash
# Test v3 with different configs
conda activate imem-v3

# Test A: Default config
export IMEM_CONTEXT_DIR=~/.context_v3_default
imem index develop --limit 100

# Test B: Modified config  
export IMEM_CONTEXT_DIR=~/.context_v3_experiment
imem index develop --limit 100
# (edit code)
imem index develop --limit 100  # Instant iteration
```

**Result:** Same code, multiple data contexts for A/B testing.

---

### C. Multiple v3 Subversions
```bash
# v3.0 (stable)
conda create -n imem-v3.0 python=3.12
# ... (install from commit b0c096e)

# v3.1 (experimental discovery)
conda create -n imem-v3.1 python=3.12
# ... (install from feature branch)

# v3.2 (HNSW backend)
conda create -n imem-v3.2 python=3.12
# ... (install from hnsw branch)
```

**Result:** Pin specific versions for comparison.
