## 7 Layers of Intelligence

### Layer 1: Template Structure
**What:** Guaranteed fields from write-time enforcement
**Intelligence:** Every chunk has Context, Solution, Rationale
**Result:** Deterministic queries, no guessing

### Layer 2: Metadata Filtering
**What:** Precise narrowing via guaranteed fields
**Intelligence:** Filter by section type, presence of rationale, timestamp
**Result:** Surgical precision without perfect ranking

### Layer 3: Graph Relationships
**What:** Explicit connections persisted and queryable
**Intelligence:** Sibling edges, genealogy chains, temporal evolution
**Result:** Traverse knowledge graph, don't recompute edges

### Layer 4: Entity Resolution
**What:** Query expansion via canonical mappings
**Intelligence:** "JWT" finds ["jwt", "JWT", "json-web-tokens"]
**Result:** Fuzzy matching independent of embeddings

### Layer 5: Schema Introspection
**What:** System describes its own capabilities
**Intelligence:** AI agents discover what's queryable programmatically
**Result:** No documentation drift, self-teaching

### Layer 6: Structured Serving
**What:** Pre-computed relationship labels in presentation
**Intelligence:** Template renders "SIBLINGS" vs "GENEALOGY" vs "TEMPORAL"
**Result:** AI comprehends relationships from structure, not inference

### Layer 7: Semantic Similarity
**What:** Fuzzy entry point into knowledge space
**Intelligence:** Maps concepts to proximity - "auth" near "JWT" near "tokens"
**Result:** Initial candidates for other layers to refine
