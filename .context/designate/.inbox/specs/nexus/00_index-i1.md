### The Generic Usefulness

**Statement 1, Line 3: "Generally useful method of creating metadata"**

**Our modular parser IS this:**

```python
# Generic template for unknown formats
class GenericMarkdownParser(SchemaAwareParser):
    def extract_section_type(self, node):
        # Just use H2 headers as-is
        # Let observer cluster variations
        header = node.content.split('\n')[0].strip('#').strip()
        return header.lower()
    
    def extract_custom_metadata(self, node):
        # Generic metadata extraction
        return {
            'word_count': len(node.content.split()),
            'has_code': '```' in node.content,
            'has_links': '[' in node.content and '](' in node.content,
        }
```
